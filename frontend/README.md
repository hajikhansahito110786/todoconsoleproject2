# Student Management Frontend

This is the frontend for the Student Management System with AI Chatbot functionality.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- User authentication
- Student management (CRUD operations)
- Todo management (CRUD operations)
- AI-powered chatbot assistant
- Audit trail showing who created/updated records

## Backend Connection

The frontend connects to the backend API running on `http://localhost:8000`. Make sure the backend is running before starting the frontend.

## Environment Variables

No environment variables are needed for the frontend, but make sure your backend is accessible at `http://localhost:8000`.
